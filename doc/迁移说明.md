# 更新说明

项目已迁移到 create-react-app 做为脚手架，迁移步骤如下

1. 创建 create-react-app 新项目
1. 展开 webpack 配置：`npm run eject`
1. 添加 less、module.less、lees 支持

    ```js
    const lessRegex = /\.less$/;
    const lessModuleRegex = /\.(module\.less|lees)$/;

    {
      test: lessRegex,
      exclude: lessModuleRegex,
      use: getStyleLoaders(
        {
          importLoaders: 3,
          sourceMap: isEnvProduction
            ? shouldUseSourceMap
            : isEnvDevelopment,
          modules: {
            mode: 'global',
          },
        },
        'less-loader',
        {
          lessOptions:{
            javascriptEnabled: true,
          }
        }
      ),
      sideEffects: true,
    },
    {
      test: lessModuleRegex,
      use: getStyleLoaders(
        {
          importLoaders: 3,
          sourceMap: isEnvProduction
            ? shouldUseSourceMap
            : isEnvDevelopment,
          modules: {
            mode: 'local',
            getLocalIdent: getCSSModuleLocalIdent,
          },
        },
        'less-loader',
        {
          lessOptions:{
            javascriptEnabled: true,
          }
        }
      ),
    },
    ```

1. 添加 babel 配置，见`babel.config.js`
1. 添加代理配置，见 `src/setupProxy.js`
1. 复制 dmp 的`src`, `resources`到项目中
1. 处理 eslint 报错
1. 禁用开发时的红色错误层
    ```js
    overlay: false,
    ```
1. 禁用 resolve-url-loader 对空 less 文件的报错

    ```js
    {
      loader: require.resolve('resolve-url-loader'),
      options: {
        sourceMap: isEnvProduction ? shouldUseSourceMap : isEnvDevelopment,
        root: paths.appSrc,
        // 添加此属性
        silent: true
      },
    }
    ```

1. 调整 html、静态 js、css、图片等资源文件
1. 添加外层文件`styleName.js`,`customeName.js`,`.gitmodules`
